Primitives, decisions, loops – Exercise
Write a program that:
1. Asks the user how many students there are.
2. Loops that many times.
3. For each student:
• Ask for total marks (integer)
• Ask for number of subjects (integer)
4. Calculate the average correctly (must use casting).
5. Print:
• The average (2 decimal places)
• The grade using if/else:
= 75 → Distinction
= 50 → Pass
< 50 → Fail
6. Print whether the student number is even or odd using %.
Constraints:
• Must use a for loop
• Must cast correctly for average
• Must not use nested if unnecessarily
• Must use modulus
Example Output
Enter number of students: 2
